# 3D Arena Shooter Project

## Installation and Solution Generation

1. Go to the `bin/` folder.  
2. Run `make.bat`.  
3. The script will generate an `ide/` folder containing `[SOLUTION_NAME].sln`.  
4. Open the Visual Studio solution.  

Then run the program through the main.

## Gameplay

The game is a **solo experience** based on a **wave system**:

- The player must survive multiple waves of enemies
- A **boss fight** occurs during the final wave

## Controls

- **Z Q S D**: move
- **Left Shift**: dash
- **Space**: jump
- **1 / 2 / 3 / 4**: switch weapons
- **R**: reload
- **F11**: toggle fullscreen
