# Projet Arena Shooter 3D

## Installation et génération de la solution

1. Aller dans le dossier `bin/`.
2. Exécuter `make.bat`.  
3. Le script génère un dossier `ide/` contenant `[SOLUTION_NAME].sln`.
4. Ouvrir la solution Visual Studio.

Lancer ensuite le programme via le `main`.

## Description

Ce projet est un **arena shooter 3D** réalisé dans le cadre scolaire.  
Il a été développé afin d’apprendre à utiliser le **moteur de jeu de l’école Gaming Campus** et à comprendre les bases du développement de jeux en 3D.

## Gameplay

Le jeu est une expérience **solo** basée sur un **système de vagues** :

- Le joueur doit survivre à plusieurs vagues d’ennemis
- Un **boss** apparaît lors de la dernière vague

## Contrôles

- **Z Q S D** : se déplacer
- **Shift gauche** : dash
- **Espace** : sauter
- **1 / 2 / 3 / 4** : changer d’arme
- **R** : recharger
- **F11** : activer / désactiver le plein écran

## Remarques

Ce projet a été réalisé à des fins pédagogiques et met l’accent sur le gameplay et l’utilisation du moteur de jeu.
